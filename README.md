# synchro
## 介绍
这是一个自用的同步项目工具箱  
项目下会有多个同步模块  
* DDNS  
* Microsoft-TODO <==> Microsoft-Calendar  
...  
* 管理界面

虽然标记为自用,若对大家有帮助我还是很高兴.

## 结构
synchro  
|-- synchro-domain  
|-- synchro-todo  
|-- synchro-manage

## 项目目标
其实很简单
- synchro-domain  
    解析检测到自家的ip变动后  
    更新到阿里域名上的DNS解析
- synchro-todo  
    获取MicroSoft-TODO上当前任务所有任务  
    将在任务变革同步到同一账号的日历上
- synchro-manage  
    这部分主要做一个页面统计  
    为上面的两个同步功能做展示
    

## 依赖
- java
- redis
- mysql
- 阿里云(域名与账号)
- 微软账号
